# Homebrew Tap

Homebrew formulae for [tinovyatkin](https://github.com/tinovyatkin) projects.

## Installation

```bash
brew tap tinovyatkin/tap
```

Then install any formula:

```bash
brew install tally
```

Or install directly without tapping:

```bash
brew install tinovyatkin/tap/tally
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [tally](https://github.com/tinovyatkin/tally) | Fast, configurable linter for Dockerfiles and Containerfiles |
